# ヒヒイロカネ・クリッカー 〜バハムートの煌めき〜

タップでバハムートを砕き、伝説の霊金「ヒヒイロカネ」を集める、かわいい放置クリッカー。

## ▶ あそぶ
**https://tktk1245.github.io/hihiirokane-clicker/**

## 特長
- 外部依存ゼロ・単一HTML（`index.html` をブラウザで開くだけでも遊べます）
- ちびキャラ＆バハムートはすべてインラインSVG描画（画像ファイル不要）
- 撃破時の金の延べ棒演出、新ステージの名乗りカットイン、BGM/SE（WebAudio）
- かわいい仲間が自動で戦う放置要素、クラスチェンジ、転生
- セーブは各ブラウザのローカルに自動保存（このページに進行は送信されません）

> このリポジトリは公開用に `index.html` のみを配置しています。
